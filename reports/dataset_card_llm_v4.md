## Known Limits

- The response candidates are LLM-generated, not human-written.
- Draft labels are generated by the same LLM generation process and were accepted
  or cleaned by Codex-assisted deep review; human audit is still recommended.
- Some instruction prompts are LLM-created because the normalized source set has
  no real instruction-following dataset.
- Bias prompts intentionally reuse some source situations to test style and
  length sensitivity, so duplicate prompt count is higher than core.
